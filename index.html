Neural Network

Python - 2018

This is my implementation of a Feed-forward Backward-propagation Neural Network. The neural network was designed to decipher basic symbols (i.e. +, -, /, etc.) from a grid of binary data. It first calculates its answer (feed-forward) then calculates the error and adjusts its calculations for the next run (back-prop). After the network has been trained on binary data, it tries to decipher symbols from analog data, based on what it has learned.
