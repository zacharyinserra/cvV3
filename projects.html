Neural Network

Python - 2018

This is my implementation of a Feed-forward Backward-propagation Neural Network. The neural network was designed to decipher basic symbols (i.e. +, -, /, etc.) from a grid of binary data. It first calculates its answer (feed-forward) then calculates the error and adjusts its calculations for the next run (back-prop). After the network has been trained on binary data, it tries to decipher symbols from analog data, based on what it has learned.

K-Means Clustering

Python - 2018

This is my implementation of the K-Means Clustering Algorithm. Several points and centroids are randomly generated. The algorithm groups the points based on the centroids. As the program updates the centroid are moved and the groups change until ideal groupings are found, at which point, the program exits.

Chord System Calculator

Java - 2018

This program solves a Chord System by finding a key value based on a number of node ID spaces and nodes within the chord. The key value is calculated by use of a finger table, which each node has, and helps to decipher which nodes in the system are possible succesors.
